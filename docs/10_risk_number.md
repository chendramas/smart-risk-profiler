## Mengapa 1–99?

### Bukan 1–100

Keputusan menggunakan skala **1–99** (bukan 1–100) didasarkan pada prinsip fundamental:

> **Angka 100 mengimplikasikan kepastian — dan kepastian tidak ada dalam investasi.**

| Argumen | Penjelasan |
|---|---|
| **Tidak ada "zero risk"** | Bahkan Deposito pun ada risiko inflasi |
| **Tidak ada "certain return"** | Selalu ada ketidakpastian |
| **Analogi matematika** | Seperti probabilitas 0 < p < 1, risk number 1 < r < 99 |
| **Standar industri** | Nitrogen Wealth (Risk Number®) menggunakan 1–99 |

### Mengapa Continuous (Bukan Discrete)?

Skala continuous memungkinkan:
- **Perbandingan yang lebih halus** antara dua investor
- **Tracking perubahan** dari waktu ke waktu (misal: dari 47 → 52)
- **Portfolio optimization** yang lebih presisi
- **Rebalancing trigger** berbasis angka pasti

## Framework Akademis

### Nitrogen Wealth — Risk Number® (1–99)

Nitrogen Wealth (sebelumnya Riskalyze) mengembangkan **Risk Number®** yang digunakan oleh >30.000 financial advisor di Amerika. Metodologi:

1. Mengukur **perceived risk** melalui questionnaire
2. Mengukur **risk required** dari goal
3. Menghasilkan satu angka: Risk Number (1–99)

### SCF 2016 — Survey of Consumer Finances (Federal Reserve)

Federal Reserve menggunakan **11-level risk tolerance measure** dalam Survey of Consumer Finances:

```
Level 1:  Takut sama sekali terhadap risiko
...
Level 6:  Moderate (seimbang risiko-return)
...
Level 11: Sangat agresif
```

Kita meng-extend dari 11 level ke **99 poin** untuk presisi yang lebih tinggi.

### Wealthfront — 100 Risk Buckets

Wealthfront menggunakan pendekatan **100 risk buckets** untuk:
- Risk scoring yang lebih granular
- Automated portfolio construction
- Tax-loss harvesting optimization

> Kita mengadopsi prinsip granularity ini dengan skala 1–99.

## Keunggulan vs Sistem 3-Kategori

### Perbandingan

| Aspek | 3-Kategori (Konservatif/Moderat/Agresif) | 1–99 Continuous Scale |
|---|---|---|
| **Presisi** | Rendah (3 bucket) | Tinggi (99 poin) |
| **Perbandingan** | Sulit bedakan investor mirip | Langsung: 52 vs 55 |
| **Rebalancing** | Threshold kasar | Trigger berbasis angka presisi |
| **Portfolio Optimization** | 3 model portfolio | 99 kemungkinan alokasi |
| **Tracking perubahan** | "Moderate → Agresif" | "52 → 67 (+15 points)" |
| **Behavioral insight** | Tidak terlihat | Perubahan kecil terdeteksi |
| **Client communication** | Subjektif | Objektif, berbasis angka |

### Contoh: Mengapa Presisi Penting

**Skenario:** Dua investor sama-sama "Moderate"

| | Investor A | Investor B |
|---|---|---|
| **3-Kategori** | Moderate | Moderate |
| **Risk Number** | 42 | 58 |
| **Perbedaan** | — | 16 poin! |
| **Alokasi ideal** | 55% FI, 45% Eq | 35% FI, 65% Eq |

Dengan sistem 3-kategori, kedua investor dapat alokasi yang sama. Dengan skala 1–99, mereka mendapat alokasi yang **benar-benar berbeda**.

## Cara Menggunakan

### Untuk Advisor

1. **Administer risk questionnaire** ke klien
2. **Hitung Risk Number** dari jawaban
3. **Cross-check** dengan Risk Need (dari Goal-Based Calculator)
4. **Build portfolio** sesuai mapping instrumen
5. **Monitor** — jika Risk Number berubah >5 poin, review portofolio

### Untuk Aplikasi

```python
def get_risk_category(risk_number: int) -> dict:
    """
    Returns category details for a given risk number (1-99).
    """
    if risk_number <= 20:
        return {
            "category": "Very Conservative",
            "color": "blue",
            "max_drawdown": -0.08,
            "expected_return": (0.04, 0.06),
            "allocation": {"fixed_income": 0.90, "equity": 0.05, "cash": 0.05}
        }
    elif risk_number <= 40:
        return {
            "category": "Conservative",
            "color": "green",
            "max_drawdown": -0.15,
            "expected_return": (0.06, 0.08),
            "allocation": {"fixed_income": 0.70, "equity": 0.25, "cash": 0.05}
        }
    elif risk_number <= 60:
        return {
            "category": "Moderate",
            "color": "yellow",
            "max_drawdown": -0.25,
            "expected_return": (0.08, 0.10),
            "allocation": {"equity": 0.55, "fixed_income": 0.35, "alternatives": 0.10}
        }
    elif risk_number <= 80:
        return {
            "category": "Growth",
            "color": "orange",
            "max_drawdown": -0.40,
            "expected_return": (0.10, 0.14),
            "allocation": {"equity": 0.75, "fixed_income": 0.15, "alternatives": 0.10}
        }
    else:
        return {
            "category": "Aggressive",
            "color": "red",
            "max_drawdown": -0.60,
            "expected_return": (0.14, 0.20),
            "allocation": {"equity": 0.90, "alternatives": 0.10}
        }
```
